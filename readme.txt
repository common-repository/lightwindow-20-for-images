== Description ==

This plugin includes lightwindow 2.0 into your wordpress and formats the anchors to images automatically.
Work's with .jpg, .jpeg, .bmp, .png, .tiff, .gif. Simply link text or thumbnails to your image that should
open in lightwindow, the plugin do the rest.

== Installation ==

1. Upload `tf_lightwindow` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
